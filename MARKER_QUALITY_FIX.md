# Исправление качества маркеров на карте

## Проблема
Маркеры локаций на карте были пикселизированными и низкого качества.

## Причина
Device Pixel Ratio (DPR) был установлен на `1.0`, что создавало маркеры с низким разрешением.

## Решение
Увеличен DPR с `1.0` до `3.0` для создания маркеров высокого качества.

### Изменения в коде:

**Файл:** `flutter_app/lib/screens/map/map_screen_new.dart`

**Было:**
```dart
// Pixel ratio — use 1.0 so markers render at logical-point size
double get _dpr => 1.0;
```

**Стало:**
```dart
// Pixel ratio — use 3.0 for high-quality, crisp markers (prevents pixelation)
double get _dpr => 3.0;
```

## Результат
- ✅ Маркеры теперь рендерятся в 3x разрешении
- ✅ Четкие, не пикселизированные иконки на карте
- ✅ Улучшенное качество текста на маркерах
- ✅ Профессиональный вид карты

## Технические детали

DPR (Device Pixel Ratio) определяет разрешение, с которым создаются маркеры:
- `1.0` = стандартное разрешение (пикселизированное)
- `2.0` = Retina разрешение (хорошее качество)
- `3.0` = Super Retina разрешение (отличное качество)

Увеличение DPR до 3.0:
- Создает маркеры размером 3x от логического размера
- Затем масштабирует их обратно для отображения
- Результат: четкие, высококачественные маркеры без пикселизации

## Тестирование
После деплоя проверьте:
1. Откройте карту в мобильном приложении
2. Проверьте четкость маркеров автомоек
3. Увеличьте/уменьшите масштаб карты
4. Убедитесь, что маркеры остаются четкими на всех уровнях зума
